### 🤖 Uso de Inteligencia Artificial

Durante el desarrollo de este proyecto, se ha usado IA para:
* Extraer el texto plano en formato HTML de la página de muestra de figma mediante OCR utilizando capturas de pantalla, para esto, el prompt especificó claramente que este no debía contener ningun tipo de estilo predeterminado, o generado mediante IA.
* Ayuda en el código para obtener una estructura HTML más limpia y semántica.
* Solucionar problemas de superposición de boxes.
* Solucionar errores o problemas de ubicación de boxes, en concreto, los iconos de footer.
